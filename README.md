# Hospital Appointment System

A full stack web application that allows patients to book appointments with doctors online.

## Technologies Used
Frontend: React (Vite), CSS  
Backend: Flask (Python)  
Database: MongoDB  

## Features
- User registration and login
- View doctors
- Book appointments
- Priority queue (normal / urgent)
- View booked appointments

## Project Structure
frontend/ → React application  
backend/ → Flask API and database connection  

## How to Run

### Backend
cd backend
pip install -r requirements.txt
python app.py

Live Demo: https://medi-connect-6bo1.vercel.app

### Frontend
cd frontend
npm install
npm run dev
